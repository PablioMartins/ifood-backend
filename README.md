# IFood Backend

## Requerimentos

-   mariadb
-   node
-   npm

## Desenvolvimento

```bash
# Clone o repositório
$ git clone https://github.com/neumanf/ifood-backend # ou seu fork

# Navegue até o diretório
$ cd ifood-backend

# Instale as dependencias
$ npm install

# Copie o arquivo .env.example para .env
$ cp .env.example .env

# Inicie o servidor
$ npm run dev
```
